# INSTALLATIONS

```bash
# install ipython-sql
$ pip install ipython-sql

# install SQL server
$ pip install pyodbc

# install MySQL
$ pip install PyMySQL

# install PostgreSQL
$ pip install psycopg2

# caching_sha2_password
$ pip install cryptography
```


OR

```bash
$ pip install -r requirements.txt
```
